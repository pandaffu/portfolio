Security Considerations

SSL Certificate

To minimise the potential security risks of the website, the most important aspect that should be taken into consideration is to prevent users data, information and to keep them safe in general. This could be done by adding SSL certificate. SSL certificate is a network protocol used for secure internet connections, making the data and sent information secure. The user using the website can be sure that the information sent will not fall into the wrong hands when a potential attack to the database occurs. Users will be more willing to fill-out the forms and make purchases thanks to the SSL certificate on the website.

2021. [online] Available at: https://www.cloudflare.com/en-gb/learning/ssl/what-is-an-ssl-certificate [Accessed 10 May 2021].

Back-ups

Another aspect is that the files and data of the website should be regularly backed-up, to prepare for the worst case scenario, such as losing all the details about the users or content of the website. These back-ups should be run every 24 hours to make sure that everything can be retrieved on daily basis if anything happens.
